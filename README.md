Impacta - Projeto integrador
Projeto integrador de uma rede social solidária com a finalidade de conectar doadores à receptores em vulnarabilidade social.

Visão Geral:
Este repositório contém uma aplicação web dinâmica, desenvolvida com Python e Flask, que integra front-end e back-end para simular o funcionamento de uma plataforma digital de impacto social.
O projeto foi concebido como parte de um projeto integrador acadêmico, aplicando conceitos fundamentais de desenvolvimento web, organização de código, separação de responsabilidades e versionamento com GitHub.

Objetivos do Projeto:
Conectar doadores a receptores em situação de vulnerabilidade social.
Incentivar ações solidárias por meio de uma plataforma digital.
Aplicar conceitos de desenvolvimento web full stack.
Desenvolver uma solução com impacto social e educacional.

Tecnologias Utilizadas:
Python — Linguagem de programação do back-end
Flask — Framework web para criação da aplicação
HTML5 — Estruturação das páginas
CSS — Estilização e layout
JavaScript — Interatividade no front-end
JSON — Estruturação e manipulação de dados
GitHub — Hospedagem do repositório

Como executar:
1- Clone o repositório:
git clone https://github.com/cibelledeas-boop/impacta_projeto_senac.git
2- Acesse a pasta: cd impacta_projeto_senac
3- Instale as dependencias: pip install flask
4- Execute o arquivo route.py
